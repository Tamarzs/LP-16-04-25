3. float e double

Os dois tipos armazenam números reais.

float
ocupa 4 bytes (32 bits) na memória.
double
ocupa 8 bytes (64 bits) na memória.

Para receber um valor do tipo double com scanf() use %lf: 

double x;

scanf("%lf", &x);

Para imprimir usando printf() use %f ou %lf: 

printf("Digitado: %f", x);

3.1. Exercício
